# EV Data Analysis

This project is based on analyzing Electric Vehicle (EV) data using Python. The goal is to clean the data, explore patterns, and visualize important trends related to EV usage and adoption.

Tools Used
- Python  
- Jupyter Notebook  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

Project File

EV_Data_Analysis_Assignment.ipynb

Analysis Includes

- Data cleaning & preprocessing  
- Exploratory Data Analysis (EDA)  
- Data visualization  
- Key insights from EV data  

## How to Run
Open the notebook using Jupyter Notebook:
```bash
jupyter notebook EV_Data_Analysis_Assignment.ipynb
